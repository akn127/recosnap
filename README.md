# RecoSnap

# 概要
休日誰かとおでかけした、話題のイベントに行った、特別なことはないけどいつものメンバーで飲みに行った・・・<br>
そんな日常の思い出を写真とテキストで思い出を投稿できる記録アプリです。

# 利用方法

### 記録投稿
1. ログイン
（ユーザー登録の場合は「新規登録はこちら」ボタンから新規登録）
2. トップページ右上の「記録する」をクリックしてください。
3. 画像添付
添付したい画像がある場合は、「ファイルを選択」ボタンをクリックし、添付したい画像を選択してください。
2枚以上添付したい場合は、直前の画像プレビュー下にある「ファイルを選択」ボタンを押すことで追加することができます。

4. 「 タイトル」入力
記録のタイトルを入力して下さい。（必須項目です）

5. 「カテゴリ」選択
どんなテーマの記録なのか、ジャンルをカテゴリ検索から選択して下さい。（必須項目です）
 
6. 「日付」入力
記録したい日付をカレンダーマークから選択、もしくはフォームに直接入力することで日付を設定できます。（必須項目です）

7.  「場所」入力
記録内容に関する場所のデータを入力します。（必須項目です）

8. 「誰といった？」の入力
一緒に行った人を記録できます。（必須項目です）

9. 「おでかけの記録」入力
自由に書ける記録スペースです。
その日あったことを日記のように記してもよし、お役立ち情報などをメモがわりに書くのもよしです。（必須項目です）

10. 「URL」入力
イベントHPのURLや、行ったお店に関しての参考URLなどを残しておきたい時に使えます。

11.  「公開・非公開」の選択
書いた記事の公開・非公開ステータスを選択できます。
他ユーザーも閲覧可能にしたい場合は「open」を選択し、自分だけで閲覧したい場合か「closed」に設定します。（必須項目です）

12. 「保存する」ボタンをクリックし、記録を保存して下さい。

### 記録の一覧を見る
1. 左上の「RecoSnap」ロゴをクリックすると、トップページに遷移します。

2. トップページでは自分の記録一覧が表示されるようになっています。

### 自分の記録の詳細表示/編集
1. トップページから、詳細表示したい記録をクリックします。
2. 画像の添付をした場合は一番上に画像が表示され、その下にテキストが表示されます。
3. 記録を編集したいときは「参考URL」の下の「編集」ボタンをクリックします。
4. 編集画面に遷移します。以前入力したテキストはそのまま残っています。画像は再度選択しなおします。
5. 「保存する」ボタンをクリックし、記録内容を更新します。

### 自分の記録の削除
1. トップページから、削除したい記録をクリックします。
2. 「参考URL」の下の「削除」ボタンをクリックします。
3. 「記録を削除してもよろしいですか？」というポップが出てきますので「OK」をクリックします。

### 他ユーザーの記録を見る
1. トップページ右上の「みんなの記録」をクリックしてください。
2. 「公開」ステータスに設定されている記録が一覧として表示されています。
3. 詳細表示したい投稿をクリックするとその投稿の詳細が確認できます。

### 他ユーザーの記録をお気に入り登録する/お気に入りを解除する
1. トップページ右上の「みんなの記録」から、お気に入り登録したい投稿をクリックして詳細を表示します。
2. 記録の右上側の「お気に入り」ボタンをクリックします。
3. お気に入りマークの色が赤く変わり、お気に入り登録されます。
4. 解除したいときは再度「お気に入り」ボタンをクリックします。
5. お気に入り登録した記録を見たい時は、ページ右上のユーザーネームにカーソルを当てた時に出る「お気に入り」をクリックします。



# 目指した課題解決
撮った写真とその日の記録がバラバラになってしまう、もしくは写真は撮りっぱなしで、詳細なことは忘れてしまった、ということがあると思います。<br>
せっかくの思い出を、記録しておけばいつでもその時の写真と詳細な記録をセットで振り返られるアプリケーションを目指しました。<br>
記録もフォーマットに従って入力するだけなので、面倒がなく、カテゴリ分けができるので旅行などの特別なお出かけだけでなく、日常の思い出も記録できます。<br>
また他ユーザーの記録を閲覧することによって次のおでかけの参考にしたり、楽しみの共有をすることができます。<br>

# 要件定義
| 機能                   | 目的                                                                                         | 詳細                                                                                             | 
| ---------------------- | -------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------ | 
| 投稿機能               | ユーザーが思い出を画像とテキストで記録する                                                   | 簡単に記録できるよう、フリーテキストだけではなく、詳細カテゴリのフォーマットは入力画面で用意する | 
| 画像投稿機能           | 詳細ページは記録テキストと写真で構成する                                                     | 最低１枚以上の写真をアップできるようにする                                                       | 
| 投稿の検索機能         | キーワードで投稿を検索できるようにする                                                       | 複数のカラム内から、検索ワードが含まれている投稿は全て表示できるようにする                       | 
| カテゴリの絞り込み機能 | カテゴリ選択で該当カテゴリの記事のみ表示させる                                               | 投稿内容のジャンルをプルダウン選択でカテゴリ指定することができる                                 | 
| お気に入り機能         | 他ユーザーの投稿をマイページに保存するため                                                   | 他ユーザーの投稿の「お気に入りボタン」を押した時に、自分のお気に入りページに投稿が保存される     | 
| ページネーション機能   | 投稿が増えた際にページ化し、見やすくするため                                                 | 投稿一覧の下部にページ送りのボタンを設置                                                         | 
| 投稿の公開・非公開機能 | 公開・非公開を設定することにより、見られたくない記事は他ユーザーからは閲覧できないようにする | 投稿を作成する際に、公開・非公開を選択できる                                                     |

# テーブル設計

## users テーブル

| Column   | Type   | Options     |
| -------- | ------ | ----------- |
| name     | string | null: false |
| email    | string | null: false |
| password | string | null: false |

### Association

- has_many :records
- has_many :favorite

## records テーブル

| Column      | Type       | Options                        |
| ----------- | ---------- | ------------------------------ |
| user        | references | null: false, foreign_key: true |
| title       | string     | null: false                    |
| category_id | integer    | null: false                    |
| date        | date       | null: false                    |
| place       | string     | null: false                    |
| with        | string     |                                |
| text        | text       |                                |
| url         | string     |                                |
| status      | integer    | null: false                    |

### Association

- belongs_to :user
- has_many :favorite


## favorites テーブル

| Column      | Type       | Options                        |
| ----------- | ---------- | ------------------------------ |
| user        | references | null: false, foreign_key: true |
| record      | references | null: false, foreign_key: true |

### Association

- belongs_to :user
- belongs_to :record
